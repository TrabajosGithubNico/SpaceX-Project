EDA with Data Visualization

This module explores the unified SpaceX dataset through comprehensive data visualization to uncover key patterns, trends, and relationships. The goal is to extract actionable insights into the factors influencing launch success, providing a deep visual understanding of the variables before proceeding to predictive modeling.

Launch Site Analysis: Visualized the relationship between flight numbers and launch sites to identify operational patterns and site-specific performance.

Payload vs. Success Rate: Analyzed how payload mass affects landing outcomes across different launch sites to identify threshold trends.

Success Rate by Orbit Type: Compared success rates across various orbits (LEO, VLEO, GTO, etc.) to determine which mission profiles pose the highest risk.

Temporal Success Trends: Extracted and plotted yearly success rates to visualize SpaceX's technological evolution and reliability improvements over time.

Correlation of Variables: Utilized scatter plots and bar charts to break down the performance of Falcon 9 boosters based on version and landing intent.

Visualization Engineering: Leveraged Matplotlib and Seaborn to create standardized, high-impact visuals that clarify complex data relationships.

Output
A comprehensive set of visualizations that identify the most influential variables for mission success. This analysis provides the data-driven justification for feature selection and helps formulate the hypotheses required for effective machine-learning modeling.
